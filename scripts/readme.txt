In order for better configuration, do below steps:

1. File Browser
	1. Always open in browser windows
		a. Select "Edit -> File Management Preferences"
		b. Select "Behavior" tab
		c. Check "Always open in browser widows" option

2. gedit Text Editor
	1. Tab width
		a. Select "Edit -> Preferences"
		b. Select "Editor" tab
		c. Set "Tab Stops -> Tab width" to 4
	2. Line numbers
		a. Select "Edit -> Preferences"
		b. Select "View" tab
		c. Check "Line Numbers -> Display line numbers"
	3. Font
		a. Select "Edit -> Preferences"
		b. Select "Font & Colors" tab
		c. Select "Font -> Editor font" as "Monospace 10"
	4.  不保存文件备份
		a. Select "Edit -> Preferences"
		b. Select "Editor" tab
		c. Uncheck "在保存文件之前创建备份文件"

3. service yum-updatesd stop
